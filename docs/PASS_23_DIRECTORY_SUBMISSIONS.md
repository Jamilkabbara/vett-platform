## Process notes

1. **Track every submission in this doc** — update Status column as
   submissions move from TODO → submitted → live.
2. **Backlinks compound** — every directory listing that DOES link
   back is a small SEO signal. Track which ones surface on Google
   Search Console as referrers.
3. **Review velocity matters more than review count** on G2/Capterra —
   3 reviews in launch week beats 10 reviews over 6 months for the
   "trending" placement.
4. **Don't pay for placement** unless ROI is clear. Many directories
   sell premium placement; default to free tier and earn organic
   ranking via reviews.
5. **Refresh quarterly** — re-check live listings every 90 days; some
   require renewal or content refresh to maintain placement.
